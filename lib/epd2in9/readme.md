# GDEH029A1 2.9" e-paper display library for Teensy 3.5

GDEH029A1 2.9" e-paper display library for Teensy 3.5

## Hardware connection (e-Paper --> Teensy 3.5)

    3.3V --> 3V3 (250 mA max)
    GND  --> GND
    DIN  --> D11
    CLK  --> D14
    CS   --> D10
    DC   --> D9
    RST  --> D8
    BUSY --> D7
